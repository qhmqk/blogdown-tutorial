checkboxInput

Last updated: Wed Jan 10, 2018

By: qhmqk in  Shiny  UI Inputs 

checkboxInput(inputId, label, value = FALSE, width = NULL)

引数

名前 説明
inputId 値にアクセスするために使用するinputのスロット
label コントロールに表示するラベル。NULLならラベル無し
value 初期値(論理値でTRUEまたはFALSE)
width ‘400px’や’100%‘などの形式で幅を指定。詳細はvalidateCssUnitを参照

値

UI定義に追加されるチェックボックスのコントロール。

説明

論理値を指定するために使用するチェックボックスを生成。

使用例

入力用のチェックボックスの状態を、renderTextにより文字列として出力します。

ui <- fluidPage(
  checkboxInput("somevalue", "Some value", FALSE),
  verbatimTextOutput("value")
)
server <- function(input, output) {
  output$value <- renderText({ input$somevalue })
}
shinyApp(ui, server)
